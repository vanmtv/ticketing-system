# 🎫 Ticketing System

A simple and efficient **ticket management system** built to handle issue tracking, service requests, or support tickets in a structured way.

---

## 🚀 Features

- Create and manage support tickets
- Track ticket status (open, in progress, resolved, closed)
- Simple and intuitive workflow
- Clean project structure for learning and experimentation
- Easily extendable for new features

---

## 🧠 Use Cases

This project can be used as a foundation for:

- Internal IT support systems
- Customer support platforms
- Bug tracking tools
- Learning project for backend

---

## 🛠️ Tech Stack

- Database: MongoDB (NoSQL)
- API: REST
- Language: Java 21 (LTS)
- Framework: Spring Boot 3.4.0

## Vulnerabilities fixes
- Jackson core vulnerability GHSA-72hv-8253-57qq
- Spring Framework vulnerable to a reflected file download (RFD)

## 📖 API Documentation & Testing
🟢 Swagger UI (Interactive Docs)
The project includes OpenAPI 3 integration. Once the application is running, you can explore and test the endpoints directly through the browser.

URL: http://localhost:8080/swagger-ui/index.html

Use case: Best for quick manual testing and viewing the JSON schemas for Ticket and Status Enums
